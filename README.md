# Locatech

Este é um projeto acadêmico desenvolvido para a disciplina de Arquitetura e Desenvolvimento Java da FIAP. O projeto ainda está em andamento e consiste em um sistema de gerenciamento de locação de veículos.

## Tecnologias Utilizadas

- Java
- Spring Boot
- Maven

## Estrutura do Projeto

- `controllers/`: Controladores da aplicação
- `entities/`: Entidades do domínio (Veículo, Aluguel, Pessoa)
- `resources/`: Configurações e recursos estáticos

## Como Executar

1. Certifique-se de ter o Java e Maven instalados.
2. Execute `mvnw spring-boot:run` para iniciar a aplicação.

O projeto está em desenvolvimento e pode conter funcionalidades incompletas ou bugs.
